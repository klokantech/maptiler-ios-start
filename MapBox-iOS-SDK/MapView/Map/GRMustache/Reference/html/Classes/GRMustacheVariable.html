Overview

A GRMustacheVariable represents a Mustache variable such as {{name}}.

You will be provided with GRMustacheVariable objects when implementing mustache lambda sections with objects conforming to the deprecated GRMustacheVariableHelper protocol.

Companion guide: https://github.com/groue/GRMustache/blob/master/Guides/variable_tag_helpers.md
